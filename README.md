# Dex++ Plugins
Plugins For [Dex++](https://github.com/AZYsGithub/DexPlusPlus)

You can find Dex++ Plugins [here](/Plugins)

## How to install plugins?
1. Download the plugin lua file
2. Go to your workspace folder
3. Copy the file to workspace/dex/plugins
4. Execute the dex script
5. The plugin should be loaded
